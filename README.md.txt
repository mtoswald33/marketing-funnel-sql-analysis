# Marketing Funnel Performance Analysis

## Overview
This project analyzes a multi-stage marketing funnel to evaluate acquisition channel performance and marketing efficiency.

## Tools Used
- SQL
- Google BigQuery

## Objectives
- Calculate overall lead-to-customer conversion rate
- Evaluate revenue performance by acquisition source
- Compute Customer Acquisition Cost (CAC)
- Calculate Return on Ad Spend (ROAS)

## Methodology
- Imported dataset into Google BigQuery
- Wrote structured SQL queries using CTEs
- Aggregated revenue and ad spend by acquisition source
- Calculated performance efficiency metrics

## Key Findings
- Google Ads generated the highest ROAS among acquisition channels.
- Conversion rates varied significantly by source.
- Some channels generated high lead volume but lower efficiency.

## Business Recommendation
Reallocate marketing budget toward high-ROAS channels while optimizing lower-performing sources to improve overall return on investment.
